**Полноценная оффлайн-система управления продажами** — без серверов, без баз данных, без интернета.  
Всё хранится локально в JSON. Работает мгновенно. Готово к использованию в реальном бизнесе.

Поддерживает **два типа магазинов** в одном приложении:
- Продажа **по штукам** (техника, одежда, книги)
- Продажа **на вес** (специи, сыпучие продукты, орехи, чай)

### Как запустить (очень просто!)

#### Способ 1 — Рекомендуемый (через лаунчер)
```bash
python main.py
Способ 2 — Напрямую
Bashpython shop_gui_classic.py    # Обычный магазин (ноутбуки, мыши и т.д.)
python shop_gui_spices.py     # Магазин специй (по килограммам)

Структура проекта
textshop-manager/
├── main.py                  ← Главный запуск (лаунчер)
├── models.py                ← Универсальные модели (Client, Product, Order)
├── shop_gui_classic.py      ← GUI для штучных товаров
├── shop_gui_spices.py       ← GUI для продажи на вес
├── file_database.py         ← Файловая "БД" (JSON, автосохранение)
├── analisys.py              ← Модуль аналитики для штучных товаров. Он выполняет преобразование заказов в табличный формат и генерацию бизнес-отчётов.
├── tests.py                 ← Основные тесты
├── test_models.py           ← Тесты валидации и логики
├── data/                    ← ← Автоматически создаётся (данные классического магазина)
├── spice_data/              ← ← Автоматически создаётся (данные магазина специй)
├── README.md                ← Этот файл
└── .gitignore

Требования (минимальные)
BashPython 3.9 или новее
Установи зависимости одной командой:
Bashpip install pandas matplotlib seaborn
tkinter уже встроен в Python — ничего больше не нужно!

Как пользоваться
Классический магазин (shop_gui_classic.py)

Добавляй клиентов
Создавай заказы: указывай ID клиента и товары (название, цена)
Все данные сохраняются в папку data/

Магазин специй (shop_gui_spices.py)

Добавляй клиентов
Создавай заказы: выбирай товар и указывай вес в кг
Автоматически считает стоимость
Данные сохраняются в spice_data/

Аналитика

Во всех магазинах есть вкладка "Аналитика"
Показывает:
Динамику заказов по дням
Топ клиентов по количеству заказов
Топ товаров по выручке или весу



Для разработчиков и энтузиастов
Хочешь добавить свой тип магазина? Легко!

Создай файл, например shop_my_cafe.py
Используй готовые модели:

Pythonfrom models import Client, Product, Order

Положи файл в корень проекта
Перезапусти main.py — твой магазин автоматически появится в лаунчере!


Тестирование
Проект покрыт тестами:
Bashpython test_models.py
python tests.py
Все тесты должны пройти успешно — это гарантия стабильности.
